# [Todo list app](https://simonportillo.github.io/todo-app/).

## Functionality
1. **Add Items**
2. **Edit Items**
3. **Drag items**
4. **Delete Items**
5. **Mark Items as Completed**
6. **Search Items**
7. **Toggle color themes**
8. **Clear Completed Items**
