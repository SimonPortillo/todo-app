# [Todo list app](https://simonportillo.github.io/todo-app/).

## Functionality
1. **Add To-Do Item**
2. **Edit To-Do Item**
3. **Delete To-Do Item**
4. **Mark To-Do Item as Completed**
5. **Search To-Do Items**
6. **Toggle Dark Mode**
7. **Clear Completed To-Do Items**
